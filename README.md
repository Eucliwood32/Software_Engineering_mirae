# 🚢 QCE (부탁해 꼬마선장!)
**"우리 배의 방향은 내가 잡는다!" Windows 전용 Python 데스크톱 팀 기여도 정량평가 앱**

*(BGM: 사카낙션 - 밤의 무희)* 🎶

인도네시아 전통 보트 경주에서 화려한 춤사위로 선원들의 타이밍을 맞추고 사기를 높이던 **'꼬마선장'** 밈을 아시나요? 
팀 프로젝트라는 험난한 항해 속에서, **QCE(Quantitative Contribution Evaluator)**는 우리 팀의 꼬마선장이 되어 줍니다! 

조장님을 위해 준비된 이 앱은 노 젓는 팀원들의 땀방울(Git 커밋, 문서 작성, 메신저 활동)을 정량적으로 평가하고 춤추듯 화려하게 시각화해 줍니다. 무임승차는 절대 꼬마선장의 눈을 피할 수 없습니다! 🕺💃

---

## 🌊 주요 특징 (4대 원칙)

- **O-1 🚣‍♂️ 삼위일체 통합분석:** Git 커밋 로그, OOXML 문서(Word, PowerPoint) 타건량, 카카오톡 메신저 활동량을 단일 파이프라인으로 통합 분석합니다. 누가 가장 열심히 노를 저었는지 한눈에 파악하세요!
- **O-2 🕵️‍♂️ 어뷰징 차단:** 꼼수를 부려 대량의 자동 생성 코드를 넣었다고요? 꼬마선장에겐 안 통합니다. Capping(1000줄 상한) 및 로그 스케일링(Log Scaling) 기법과 통계적 이상 징후 탐지로 어뷰징을 원천 차단합니다. 이상 신호는 툴팁과 카드로 조장에게 슬쩍 귀띔해 줍니다.
- **O-3 📊 춤추는 시각화 및 리포트:** 3-스크린(제출-로딩-결과) 아키텍처를 기반으로 막대, 레이더, 산점도 차트로 기여도를 시각화합니다. 가중치 슬라이더를 통해 개발/기획/균형 등 원하는 100% 비율로 유연하게 지휘할 수 있습니다! (리포트는 `.md` 및 `.csv`로 저장 가능)
- **O-4 🔒 망망대해에서도 로컬 실행:** 네트워크 통신 일체 금지(0byte)! 외부 서버나 LLM API 없이 오직 로컬 CPU/RAM만 사용하여 우리 배의 소중한 데이터를 완벽하게 보호합니다.

| 이름 | 역할 | 주요 담당 모듈 |
| :--- | :--- | :--- |
| **조원희** | Backend | OOXML 파서, Git 분석, 메신저 파서, 인코딩 처리 |

## 🛠️ 기술 스택 (배의 설계도)
- **언어:** Python 3.10+
- **UI:** PyQt6 (Strict MVC 아키텍처, View ↔ Controller 완벽 격리, Worker Thread 적용)
- **시각화:** matplotlib (BaseChartWidget 기반 막대, 레이더, 산점도 애니메이션)
- **파싱:** python-docx, python-pptx, csv, subprocess
- **NLP:** kiwipiepy, soynlp (Pure Python)
- **빌드:** PyInstaller (`--onefile` 단일 실행 파일)

## 🚫 제약 사항 및 의존성
- **환경:** Windows 10/11 x64 전용 (파도치는 바다 말고 윈도우에서 실행하세요)
